Specifications to run this program:

1. Any OS with JAVA support( preferably Windows)
2. JDK latest version after 1.6. For downloading JDK follow this link:
http://www.oracle.com/technetwork/java/javase/downloads/index.html
3. Open Command prompt or terminal and go to the directory where codes are stored in terminal
4. Give path for Java compiler to compile code. In windows it can be done by
set path="C:\Program Files\Java\jdk1.7.0_45\bin"
5. for compiling code run following commands
javac IS_ngui.java
javac Client_UI.java (Ignore any warnings but not errors)

6. For running code run following commands:
java IS_ngui
java Client_UI
